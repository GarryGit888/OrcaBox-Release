# Release Notes

## 1.1.2 - 2026-07-21

完成 OrcaBox 1.1.2 macOS Apple Silicon 与 Windows x64 发布包。

### 新增与改进

- 新增独立 macOS Intel 打包配置与构建支持。
- 完善素材管理、媒体预览、搜索、设置、批量操作与任务中心工作流。
- 增加导入队列、审核面板、搜索中心、操作历史与资料库备份能力。
- 优化图片裁剪、Lottie、视频预览和本地媒体处理路径。
- 同步浏览器扩展、版本文档、官网发行索引和分发检查脚本。

### 验证与已知说明

- 94 个测试文件、719 个测试全部通过。
- macOS DMG、ZIP 与 Windows EXE、blockmap 均已生成并完成资源完整性检查。
- macOS 安装包未使用 Developer ID 签名，尚未完成公证。
- Windows 安装包由 macOS 主机跨平台构建，未包含 Windows 系统分享原生插件；系统分享能力在该包中不可用。

## 1.0.0 - 2026-07-17

首个公开发行版本。

### 功能

- 本地素材目录索引与实时文件监听
- 多格式缩略图、元数据和沉浸式预览
- 图片、音频、视频、Lottie、3D 模型、字体和文档支持
- 标签、合集、评分、搜索与批量管理
- 本地 AI 标签分析、相似图片和颜色搜索
- macOS Apple Silicon 与 Windows x64 安装包

### 已知说明

- macOS 安装包为 ad-hoc 签名，尚未完成 Apple Developer ID 公证。
- Windows 首次安装时可能显示 SmartScreen 提示。
- 此公开仓库仅用于产品介绍、问题反馈和安装包分发，不包含应用核心源代码。
